### Prerequisites
- C++ compiler (g++, clang++, or any C++11/14/17 compatible compiler)

### Compiling and Running

To compile any C++ file:
```bash
g++ -std=c++17 -o output_file filename.cpp
./output_file
```

Example:
```bash
# Compile Two Sum problem
g++ -std=c++17 -o twosum DSA/arrays/TwoSum.cpp

# Run the compiled program
./twosum
```

### For Windows Users
```bash

output_file.exe
```

If you're new to DSA, here's a recommended order to explore the problems:

1. **Start with Arrays**: Basic operations, searching, and sorting
2. **Move to Recursion**: Understand recursive thinking and base cases

4. **Practice Strings**: String algorithms and pattern matching
5. **Advanced Topics**: Backtracking, dynamic programming, and optimization
